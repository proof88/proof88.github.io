Detailed Description

Contains PRRE shared settings.

These settings are used by a few classes, but we store these in one place (here), to avoid redundancy and inconsistency. Other classes reach these settings through pointers. One class may be responsible for properly initializing a setting, while other class may be responsible for providing get/set access to the user. For a user, this class is not known. Users can manipulate shared settings through public classes. This class is singleton, of course. IMPORTANT: the main PRRE object should initialize this class before any other class attempts to use it.
