Detailed Description

Incremental fixed-function hardware renderer path.

Only deviations from the original PRREIRenderer documentation are mentioned here. This renderer has multiple rendering paths, controlled by the render hints. The default render hints for this renderer is DefaultHintsForHWfixedPipeRenderer.
