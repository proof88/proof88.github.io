◆ getUtiliser() [1/2]

PRREManaged * PRREManaged::getUtiliser ( )

Returns the managed that utilizes this managed.

The utiliser is another PRREManaged instance, usually having different class, and that class becomes complete with the utilised managed instance. For example: every PRREMesh3D instance has an associated PRREMaterial instance, in this case PRREMesh3D instance utilizes a PRREMaterial instance, so the latter instance's getUtiliser() returns the PRREMesh3D instance. A PRREManaged instance can have multiple utilised PRREManaged instances, but any PRREManaged instance can have maximum 1 utiliser instance.

Returns
The managed that utilizes this managed.
